Smart Collections

Automatically creates movie collections for series (using TMDb collection metadata already on your movies) and for custom themes (e.g. Christmas movies). Collections this plugin creates are self-healing: if one disappears it will be recreated on the next sync — unless you intentionally deleted it, in which case it will never be recreated.

Series Collections

Groups movies that share the same TMDb collection name (e.g. "John Wick Collection") into a Jellyfin collection.
A collection is only created when at least this many movies belong to the series. Default: 2.
