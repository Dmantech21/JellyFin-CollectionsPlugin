Smart Collections

Automatically creates movie collections for series (using TMDb collection metadata already on your movies) and for custom themes (e.g. Christmas movies). Self-healing: if a plugin-managed collection disappears it is recreated on the next sync — unless you intentionally deleted it (tracked below).

Series Collections

Groups movies that share a TMDb collection name (e.g. "John Wick Collection").
Only create a collection when at least this many movies belong to the series. Default: 2.

Schedule

How often the sync runs automatically. Trigger manually from Administration → Scheduled Tasks → Smart Collections Sync.

User-Deleted Collections

The plugin automatically tracks when you delete a collection it created and will not recreate it. Remove an entry here to allow the plugin to recreate it on the next sync.
